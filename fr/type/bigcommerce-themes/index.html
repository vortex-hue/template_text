Foire aux questions sur la conception de modèles BigCommerce

Les thèmes réactifs BigCommerce sont-ils bons ?

Ils sont une bonne option pour le commerce électronique. Vous pouvez facilement les modifier pour créer une boutique en ligne unique et moderne avec de nombreux produits. Ils sont faciles à gérer et à tenir à jour. Les conceptions réactives offrent aux utilisateurs d'appareils mobiles la meilleure expérience. Tout fonctionne correctement sur les appareils avec différentes tailles d'écran.

Comment choisir le meilleur design de site BigCommerce chez TemplateMonster ?

Utilisez le panneau de gauche pour définir les sujets, les balises, le jeu de couleurs, les fonctionnalités, le style, le prix, etc. Il filtre les produits qui ne correspondent pas à vos critères. Lorsqu'il vous reste quelques options, ouvrez la page de détails pour en savoir plus à leur sujet. N'oubliez pas d'utiliser la fonction de démonstration en direct pour voir à quoi tout ressemble et fonctionne.

Que sont les modèles premium Stencil BigCommerce ?

Stencil est le moteur de thème utilisé par BigCommerce. Les modèles construits sur Stencil sont compatibles avec le constructeur de page. Ces produits ont une catégorie distincte sur le marché TemplateMonster. Veuillez consulter la liste de nos modèles de pochoir disponibles . Ils sont une option parfaite pour les personnes qui ne sont pas familiarisées avec le codage.

Comment personnaliser la conception d'un modèle BigCommerce ?

La première option consiste à utiliser un constructeur de page. Il faut aller dans le menu vitrine, puis choisir mes thèmes et personnalisation. Il ouvre le constructeur de page et permet de modifier divers éléments.

La deuxième option consiste à modifier le code. Allez dans le menu vitrine, puis choisissez mes thèmes et cliquez sur le bouton avancé et sélectionnez faire une copie. Nommez-le et attendez - la duplication des fichiers peut prendre un certain temps. Lorsque le processus est terminé, cliquez sur l'icône d'action et modifiez les fichiers de thème. Vous pouvez également accéder à l'édition de fichiers à partir du constructeur de pages. Vous devez sélectionner les styles de thème et choisir de modifier les fichiers de thème.

Il existe également des thèmes de plans hérités. Pour les modifier, vous devez vous rendre dans le menu vitrine, puis sélectionner mes thèmes et appuyer sur le bouton modifier HTML/CSS.
