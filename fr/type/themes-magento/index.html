Caractéristiques principales


Responsive

La natute entièrement responsive des thèmes Magento fournit une excellente expérience utilisateur peu importe la résolution d’écran de votre dispositif. La taille d’écran n’affectera pas l’excellente représentation de votre page web, et donc les utilisateurs profitrtont du design exceptionnel de votre site Web. Chaque elemnt du site Web, y compris le slider des images, est pleinement responsive. Cela garantit le meilleur rendement dans n’importe quelles circonstances.

Documentation appropriée

Tous les thèmes sont fournis avec des instructions concises de démarrage rapide pour l’installation et la modification du modèle. La documentation comprend les extensions nécessaires qui ont été utilisés dans le modèle ou peuvent être utilisés en cas de besoin. Gardez-la à portée de main pour référence. Ces manuels utiles permettent de personnaliser votre conception pratiquement sans efforts.

Traduction (Localisation)

Nos modèles Magento sont traduits dans 4 langues: Anglais, Allemand, Espagnol, et Russe.

Aperçu des catégories en liste ou en grille

Différentes façons d’aperçu pour la page fiche de produits sont disponibles à vos visiteursdans nos modèles Magento. Ils peuvent choisir la façon qu’ils aiment le plus.

Carousel de produits responsive (touch)

Les thèmes Magento offrent aux clients la possibilité de voir les produits dans un carrousel. Vous pouvez facilement organiser vos images de produits dans une galerie responsive avec la possibilité de parcourir les photos du bout des doigts sur les smartphones et les appareils portables.


Polices d'icônes (Retina Ready)

Toutes les icônes sont remplacées par les polices d’icônes (l’exception font les icônes d’étoiles pour l’évaluation des produits). En étant vectorielles par nature, les polices d’icônes sont croquantes et évolutives. Soyez sûr d’obtenir la qualité de prestation sur les écrans avec un PPI différentю
