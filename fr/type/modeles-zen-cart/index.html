Caractéristiques


Documentation appropriée

Les modèles ont une documentation appropriée. Un manuel compréhensible vous guidera à travers le processus d’installation de la plate-forme et du modèle, et en outre la personnalisation du site. Avec toutes ces instructions minutieuses vous procédez très vite.

bxSlider Integré

La particularité de ce slider réside dans le fait que tous ses paramètres peuvent être modifiés via le panneau d’administration. Cela simplifie considérablement le processus de création de la bannière.

Module des catégories déroulantes

Les catégories sont affichées horizontalement dans l’en-tête. Maintenant, il y a aussi la possibilité d’afficher les sous-catégories en une ou plusieurs colonnes, avec ou sans images.

Module des icônes sociales

Les icônes des médias sociaux (Facebook, Twitter, Google+, Pinterest) sont affichés sur la page du produit permettant aux gens de partager leurs produits préférés avec leurs amis.

Bannières HTML

Il n’y a aucune option pour la création de la bannière dans un Zen Cart standart. Cette fonctionnalité est implémentée dans nos modèles. Cela vous donne l’occasion de faire votre propre bannière selon les exigences du client.
