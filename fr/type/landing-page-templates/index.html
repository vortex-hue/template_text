Caractéristiques principalesPour votre business pour créer des pages d’aterrissage de la meilleure façon


Design adaptatif

Etre responsive est incontournable de nos jours, soyez sûrs que votre contenu sera délivré en sa totalité à tout dispositif, que ce soit un smartphone ou un ordinateur de bureau. Tous les élement essentiels des modèles de pages d’aterrissage y compris le slider, le formulaire de contact et le menu sont conçus pour être 100% responsive pour s’intégrer parfaitement dans toute fenêtre de visualisation possible.

Bootstrap

Bootstrap donne beaucoup d’extras tout en le rendant facile à comprendre les guides de style. Il vient aussi avec des fichiers SASS pour tout CSS donc vous pouvez changer facilement les couleurs, les polices, etc.

Compatibilité avec différents navigateurs Web

Avec une combinaison infinie de navigateurs modernes et anciens, il est de notre responsabilité de développer des modèles de sites Web qui fonctionnent de manière adéquate sur tout navigateur possible.

Intégration des icônes vecteur et Web Fonts

Chaque package de modèle inclut un ensemble de Font Icons qui sont une parfaite solution pour tout site Web. Font icons sont des icônes vectorielles adaptables qui peuvent changer facilement de taille et de couleur, sans perte de qualité.

Fichiers PSD inclus

Tous les fichiers PSD sont inclus dans le package de modèle. Ils sont en couches et organisés, pour faciliter votre travail avec eux.

Adjonctions de TM

Le modèle possède une boîte de recherche entièrement fonctionnelle, des formulaires de contact et d’inscription. Dans l’ensemble de documentation du modèle vous trouverez les instructions sur leur utilisation et personnalisation.

Documentation

Chaque modèle est livré avec une documentation appropriée qui vous aidera à installer rapidement le modèle, et vous fournira de précieux conseils sur la personnalisation et la maintenance.
