Caractéristiques principalesModèles OpenCart ont une matrice de fonctionnalités viables pour le commerce électronique


100% adaptatifs

Design complètement adaptatif est fait pour offrir une bonne performance sur tous les appareils, peu importe leurs résolutions. Les utilisateurs peuvent profiter d’un look branché de votre site Web sur toute taille d’écran. Pour produire une
expérience utilisateur optimale pour différents environnements, le slider et d’autres éléments du site sont créés pour être absolument adaptatifs.

Bien documenté

La documentation incluse dans le package de téléchargement du modèle est absolument exhaustive. Un manuel rapide va vous aider à installer la plateforme OpenCart et le modèle, puis personnaliser votre site Web. Grâce à des instructions d’experts vous finirez le travail en un clin d’œil.

Mobile Sidebar Menu

Une fonctionnalité tout à fait pratique pour la navigation sur un site web sur les appareils mobiles. À l’aide d’un menu mobile qui apparaît en glissant de gauche à droite, vous obtenez un accès rapide aux liens et sections principales du site.

Vqmod intégré

Le principe de fonctionnement de Vqmod est le suivant : au lieu de modifier les principaux fichiers de base, le mod donné permet d’enregistrer toutes les modifications sur le fichier externe. Ce fichier est traité alors que le
chargement et les modifications sont insérés dans le code lors de ce processus.

Lazy Load

Lazy load est le délai de chargement des images sur des longues pages Web. Les images à l’extérieur de la zone de visualisation ne seront pas chargées jusqu’au moment où l’utilisateur fait défiler sur eux. En utilisant Lazy Load sur les pages
longues fera le chargement de la page plus rapide.

Design

Stick-to-Top Menu

Le thème est fourni avec un menu qui reste collé en haut de la page lorsque vous la faites défiler. Cela permet de maintenir le menu de navigation principal devant les yeux des utilisateurs tout le temps, alors ils ne doivent pas
refaire tout le chemin pour se rendre à une autre page.

Défilement en parallaxe

Application de défilement en parallaxe peut vraiment aider à engager vos visiteurs. Les gens se sentent automatiquement plus intéressés lorsqu’ils font défiler la page, parce que cet effet modifie la profondeur et le contenu de leur point de vue. C’est un truc psychologique qui va sûrement tirer votre visiteur plus profondément dans le site Web et enflammera son désir d’explorer.

Bloc personnalisé

Bloc personnalisé est une zone où vous pouvez insérer des informations de différents types, le formulaire de contact, les modalités de vente et les détails de paiement. Chaque fois qu’un nouveau produit est ajouté, ces données restent
automatiquement en place.

Carousel

Vous pouvez présenter un grand nombre de produits avec cette galerie à droite sur la page d’accueil. La présentation aura l’air vraiment organisé et élégant, ce qui est important car beaucoup de clients sont réticents à creuser plus profond que la page d’accueil de votre magasin, surtout quand il semble confus.

Vidéo d’arrière-plan

Magnifique vidéo en arrière-plan fait que les gens s’arrêtent et la regardent. C’est une occasion parfaite pour montrer vos meilleurs produits d’une façon interactive.

Facebook Widget et Google map


Facebook Widget

Facebook widget aide à vous fournir le moyen le plus simple et le plus attrayant pour afficher vos articles Facebook ou les likes dans la barre latérale. Il est très facile à configurer, il vous suffit d’ajouter l’URL de votre page
facebook.

Google Map

Le placement de la carte Google sur la page de votre site donne un indice à vos visiteurs sur votre emplacement physique. Cette approche aide l’utilisateur à connaître l’emplacement de votre entreprise et les heures de travail. Ils n’ont plus
besoin de rechercher cette information sur d’autres pages.

Twitter Widget

En utilisant Twitter feed widget vous pouvez afficher vos propres messages Twitter qui sont hashtaggés avec un certain mot-clé sur votre site. Pour commencer, vous devez vous connecter à votre compte Twitter ou spécifier le hashtag. C’est sûr que cela va vous aider à accroître l’activité sur votre profil Twitter.

Pinterest Widget
