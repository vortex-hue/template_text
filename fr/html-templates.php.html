Responsive

Design absolument responsive offre une excellente expérience sur tous les appareils, peu importe les résolutions qu’ils soutiennent. Tout le monde peut apprécier l’apparence fabuleuse de votre site Web sur toute taille d’affichage. Le slider et les autres composants sont faits pour être 100% responsive pour produire une expérience utilisateur optimale pour différents environnements.

Documentation appropriée

Excellente documentation est attachée au produit. Ces brefs manuels donnent des instructions détaillées sur la façon d’installer et de personnaliser le modèle. En outre, vous pouvez utiliser ces informations pratiques pour trouver les caractéristiques de ce produit, ainsi que les plugins qui sont utilisés ou peuvent être utilisés dans le modèle.

Intégration des icônes vectorielles et des Web Fonts

Font Awesome icon font est inclus dans les modèles de sites Web. Vous pourrez personnaliser instantanément la taille, la couleur, l’ombre portée et tout ce qui peut être fait avec CSS. Les modèles présentent aussi la variété des polices grâce au service Google WebFonts. Ceci permet de choisir la police que vous voulez pour votre site web.

Bootstrap

Certaines parties de nos modèles HTML sont construites avec Bootstrap framework afin que tous les options de style, scripts et composants Bootstrap soient disponibles pour vous.

Compatibilité avec différents navigateurs Web

Nous garantissons que nos modèles seront affichés correctement dans tous les navigateurs modernes, y compris Google Chrome, Internet Explorer, Mozilla Firefox, Opera et Safari.

TM Add-ons

Presque tous les modèles ont une boîte de recherche fonctionnelle, les formulaires de contact et d’abonnement. Vous trouverez les instructions sur la façon de les utiliser dans la documentation fournie avec le modèle.
