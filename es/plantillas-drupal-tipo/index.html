Plantillas Drupal

Plantillas Drupal de Template Monster son temas que Usted puede instalar en esta plataforma CMS para personalizar el diseño de su sitio web ya existente o incluso para crear página web nueva. Drupal es una plataforma CMS de código abierto que ofrece a webmasters un conjunto grande de opciones administrativas y, al mismo tiempo, es muy fácil de usar.

Ahora Usted puede probar este tipo de producto antes de comprarlo! Plantilla Drupal 7.x Responsive Gratis le da esta oportunidad.

Todas Características


Diseño Responsive

Diseños Responsives garantizan una experiencia de usuario fantástica en todos los dispositivos móviles con varias resoluciones de pantallas. Ahora el tamaño de pantalla no importa! Todos los componentes esenciales (incluso slider) de la plantilla son 100% adaptativos!

Amplia Documentación

Le ofrecemos la documentación detallada para su plantilla. Incluso el manual paso a paso de instalación de Drupal, de plantilla, y de personalización del sitio. Con estas instrucciones todas las etapas del trabajo con nuestro producto serán relativamente fáciles!

Instalación/Exportación/Importación Fácil

Hemos añadido los archivos de instalación de plantilla al paquete de descarga. Después del inicio rápido, su plantilla obtendrá el mismo aspecto que en la demostración en vivo. Todos los procedimientos son muy simples y claros, así que Usted podrá agregar fácilmente el contenido necesario al sitio web.

Portafolio con Filtros

Esta opción le permite aplicar filtros por categorías o etiquetas. Elija entre 3 diseños de portafolio mediante Views Isotope, establezca el número de entradas mostradas en la página, o personalice el diseño de cada entrada por separado. Se permite aplicar la opción de filtro a todas las alternativas de portafolio.

Gestión de Datos (Copia de Seguridad Avanzada)

Actualice rápidamente el framework, haga una copia de seguridad del framework o de un solo tema, utilizando el sitio de Demostración. Puede guardar los archivos de copia de seguridad para utilizar ellos más tarde. Todos los procedimientos de actualización y de creación de copia de seguridad son muy fáciles de realizar.

¿Qué es Drupal?

Drupal es un sistema de gestión de contenidos (CMS) (de código abierto) que se basa en PHP y que también se considera un marco modular. Drupal permite a sus usuarios crear y organizar contenidos web, editar diseños, automatizar algunas funciones administrativas, etc. A pesar de su interfaz diversa, Usted podrá realizar todas las funciones sin conocimientos de programación. Algunos expertos consideran Drupal un “framework de aplicación web” debido a su amplia gama de servicios y funciones. Hoy en día CMS Drupal es uno de los sistemas de gestión de contenidos más avanzados disponibles en la Web. El conjunto de opciones de este CMS satisface las necesidades aún más sofisticadas del mantenimiento de sitios web.
