Plantillas VirtueMart

Plantilla VirtueMart es un tema desarrollado especialmente para el software ecommerce VirtueMart Joomla. TemplateMonster.com les ofrece su colección de Temas VirtueMart que son plantillas desarrolladas a base de Joomla que les permiten ejecutar sitios web de comercio electrónico. Esto significa que ellos son útiles tanto para sus clientes que utilizarán ellos para comprar productos como para Usted. Los VirtueMart templates de TemplateMonster son diseños profesionales creados por los diseñadores eCommerce experimentados. Nuestras Plantillas VirtueMart le ayudarán a crear sitios más funcionales y atractivos.

Esta Plantilla Gratuita para VirtueMart 2.0.22 es un diseño de muestra que Usted puede descargar y probar de utilizar.

¿Qué es Plantilla VirtueMart?

Plantilla VirtueMart es un diseño para comercio electrónico, creado específicamente para VirtueMart que es una extensión de carrito de compras de Joomla.

VirtueMart español es una extensión gratuita ecommerce de código abierto para el sistema de gestión de contenidos (CMS) Joomla. Esta extensión es desarrollada a base de PHP y se implementa con el entorno de base de datos MySQL. El plugin VirtueMart le permite crear un número ilimitado de productos y categorías, vender productos descargables, utilizarlo en un modo de catálogo, usar una variedad de métodos de pago y precios múltiples para un solo producto y mucho más.

Características


Diseño Responsive

Diseños son totalmente adaptativos y garantizan el rendimiento excelente en todos los dispositivos móviles, no importan las resoluciones de pantallas que tienen. Toda la gente puede disfrutar con el aspecto fabuloso de su sitio web en pantallas de cualquier tamaño. Su slider y otros componentes son 100% adaptativos.

Búsqueda Ajax

Este módulo permite a los usuarios ver los resultados de búsqueda de productos sin tener que recargar la página. Esta opción garantiza una experiencia de búsqueda agradable.

Módulo de Slideshow

Los desarrolladores han ampliado la funcionalidad de este módulo para que Usted pueda adaptarlo a sus propias necesidades. Le permite elegir nuevas imágenes para slider, cambiar descripciones, o añadir enlaces.

Módulo de Banners

Este módulo permite configurar, eliminar o cambiar el aspecto de banners directamente en el panel de control.

Documentación Detallada

La documentación incluida en cada paquete de plantilla es muy detallada. Las instrucciones paso a paso le ayudarán a instalar VirtueMart, así como instalar y personalizar su plantilla. Con las instrucciones minuciosas Usted no tendrá problemas con estas tareas.

Carrito Ajax

Este módulo permite a los usuarios revisar los productos agregados al carrito y su cantidad sin tener que recargar la página. Además, el módulo ofrece una opción de eliminación rápida de productos del carrito.

Carrusel de Productos

Le permite mostrar imágenes de productos en una galería carrusel. También hay opciones para editar y mostrar precios, asignar imágenes de productos, añadir títulos, botones, y descripciones a sus productos.
