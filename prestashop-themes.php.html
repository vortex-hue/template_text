PrestaShop platform has plenty of modules meant to expand its initial functionality. These highly versatile extensions are used to customize the web pages of an online store. Have a look at the popular PrestaShop modules you can apply in your online store:

TM Mega Layout moduleallows you to customize the layout for 3 sections (hooks) on the main page of the website and create layouts of your own.

TM Category Products module is meant for displaying single products, products from a category or all products.

TM Mega Menu module allows creating complex menus. It also gives you the ability to display one item (without sub-menu), a menu item with a sub-menu and a menu item with sub-menu by means of a multi-row and multi-column block.

TM Advanced Filter module allows doing a quick product search based on its features. Also, it allows creating filters with the unlimited number of search criteria, available in the store by default or are created, when adding the products.

TM HTML Content module allows embedding HTML content (image, HTML, links, etc.) It is used to expand design opportunities and add advertisement/information blocks to your store.

TM Lookbook module will help you create lookbooks and put them together to make collections. Lookbooks will let you see the products in real life or directly on a model.

TM Related Products module will display similar or related products to those your clients were looking for on the page.

TM Social Feeds module incorporates social networks feeds into your website, so that the site visitors would see your latest social news and posts.

PrestaShop Templates FAQ

What is PrestaShop?

PrestaShop is a freemium, open-source software used to build an online store. It has gained popularity as an eCommerce solution used by small and large businesses worldwide. It currently powers 300,000 shops worldwide and is available in 60 different languages.

How to install the PrestaShop theme?

Installing the theme should not be an issue, especially if you've chosen the theme of the latest 1.7 version that is easier to use compared to its predecessor. You can install the theme either manually or using PrestaShop Cloud or Themeinstallator.

When proceeding with manual setup, download the theme and repack the zip file you've just downloaded. Go to the PrestaShop admin panel and then head to Preferences > Themes. Select the recently downloaded theme, click on the Advanced Settings, and save.

To install the theme through PrestaShop Cloud, open the admin panel, and navigate to Preferences > Themes. Select "Add New Theme" and type in your add-on details. Click on "Use This Theme."

To set up the template with Themeinstallator, start with installing it. Then, go to the "Import from Your Computer" section and push the Browse button. Finish with uploading the theme.

How can I effectively edit my PrestaShop theme?

If you want to make some minor changes or tweaks to your PrestaShop theme, here are a few tips on where to start. Use developer tools in Google Chrome to detect the element you are searching for and find the file that needs edits. To change colors and basic layouts, go with a custom.css file, whereas grid/list switcher will let you switch between grid or list layouts manually. Also, use an editor, such as Notepad+ (or TextWrangler for Mac users), for quick changes.

How to edit email templates in PrestaShop?

When your customers create an account on the site, subscribe to a newsletter, make a purchase, receive payment and shipment confirmation, they get default email templates. But if you want to make these emails sound unique and match your corporate identity, you can modify them through the admin panel. For this, log in and navigate to Improve > International > Translations. Under the Modify Translations block, select certain positions from dropdowns and click Modify. You will be redirected to email templates where you can select the one to edit via Edit HTML version or View/Edit TXT version tabs.
