How to Edit Content

There are several popular actions that WordPress users or marketers can do with the help of the plugin discussed in this article. Note, all steps are very simple to make:

  • Install the plugin. Buyers don't need to consider this issue when ordering Gutenberg templates from TemplateMonster. After all, web developers added the plugin to the theme. If you want to install it separately, you should know that installation and activation are no different from other WP plugins. So, there'll be no problem coping with the task.
  • You can add a new record by clicking the Records section and the Add New button.
  • To add text to a Record, you should copy it from the file where you have it stored and paste it into blocks. Remember, the plugin provides for the division of the page into separate blocks. Therefore, you may think about how to fill the texts in individual parts. For example, divide them into subheadings.
  • There are two ways to add one more block. It's necessary to press the button with a plus sign, which is located under existing blocks. The second option is to click the same button at the screen top. In this case, a new block appears at the very end.
  • To move a block, select it by simply clicking once. You'll see an arrow on the left side; by clicking that, you choose the option to move - up or down.
  • Adding a button to an article is also possible. To do this, you need to insert an additional block Button. At the same time, be sure to fill in the fields where you enter the text displayed on the button and the URL where the visitor goes by clicking on the button.

As you can see, creating pages is quite easy. You just need to take some time to learn the process.
