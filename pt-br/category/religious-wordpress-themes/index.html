Como alterar o nome do tema do Church WordPress?

Siga as etapas a seguir para renomear 'themeXXXXX' para um nome personalizado no WordPress.

  • Navegue até Aparência> Editor
  • no arquivo de folha de estilo (style.css), altere o Nome do tema: themeXXXXX para o seu nome.
