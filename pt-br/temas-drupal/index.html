O que é Drupal?

Drupal é um sistema de gerenciamento de conteúdo open source baseado em PHP (CMS) que também é considerado um modulo framework. O Drupal permite aos usuários criar e organizar o conteúdo do site, alterar o design, automatizar algumas funções administrativas, etc. Apesar de sua interface diferente, você pode executar todas as funções sem habilidades de programação. Alguns especialistas vêem Drupal como um “framework de aplicações web” para o seu ampla kit de serviços e funções. Hoje Drupal CMS é um dos mais avançados sistemas de gerenciamento de conteúdo disponíveis na web. O conjunto de funções oferecidas por este CMS atende às mais sofisticadas manutenções de websites. No entanto, é sempre conveniente e fácil de usar, graças à organização da interface inteligente. Elabore Um Belo Website em 3 Minutos clicando aqui criar sites.
