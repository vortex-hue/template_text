Caracteristicas


Twitter Bootstrap

Os desenvolvedores usaram a terceira versão de um framework popular para dar mais flexibilidade aos nossos temas Shopify quando se navega em dispositivos diversos.

Mudança De Cores via Painel de Сontrole

É possível alterar as cores do plano de fundo e outras usadas no modelo através do painel de administração. Nós usamos arquivos gráficos pequenos em nossos temas, que os tornam leves para download e conveniente para a substituição dos esquemas de cores.

Flexibilidade Das Configurações Dos Blocos

O painel de controle permite que você defina o seu logotipo, slogan, blocos de texto, links, imagens, títulos, texto no slider e banners, o número de produtos apresentados, contatos e outros elementos do template.

Mudanças de Estilo Via Painel De Controle

As fontes nos templates são fáceis de modificar com o painel de administração. Você pode alterar o tipo de família, o tamanho e as cores da fonte usadas. Esta opção está disponível para o texto principal, títulos, cabeçalhos, nomes de produtos e preços.

Simplicidade E Grande Capacidade Do Painel De Controle

Não é necessário nenhum conhecimento de CSS ou programação HTML para modificar o nosso template. A maioria das opções são simples e claras para os usuários. Você obtém uma ferramenta poderosa para a sua loja online.

Conversor Pratico De Moedas

Não há mais necessidade de recarregar a página para alterar a moeda. Os preços são convertidos imediatamente após a seleção da moeda desejada.
