O que é VirtueMart?

O Modelo VirtueMart é um projeto de e-commerce (comercio eletrônico), criado especificamente para ele, VirtueMart é uma extensão de carrinho de compras Joomla. VirtueMart é uma extensão open-source de ecommerce para o sistema de gerenciamento de conteúdo (CMS) Joomla. Esta extensão é baseado em PHP. desenvolvida e implementado com banco de dados MySQL. O plugin VirtueMart permite criar um número ilimitado de produtos e categorias, vender produtos para download, coloca-los em um modo de catálogo, usar uma variedade de métodos de pagamento, múltiplos preços para um único produto e muito mais. Características


Design Flexível

Você vai ter um site que se adapta adequadamente ao tamanho de qualquer tela e qualquer dispositivo, ao vez de esconder elementos. Além disso, este projeto garante uma experiência do usuário inesquecível.

Busca Ajax

Este modulo permite os usuários verem resultados de busca de produtos sem ter que carregar a pagina. Está opção garante uma expêriencia de busca agradavel.

Módulo de Slideshow

Os desenvolvedores ampliaram a funcionalidade deste modulo para que você possa adapta-lo a suas próprias necessidades, ele permite escolher novas imagens para o slider, trocar descrições e links.

Módulo de Banners

Este modulo permite configurar, eliminar ou trocar o aspecto de banners diretamente no painel de controle.

Documentação Detalhada

A documentação incluída nos templates, são muito detalhadas. As instruções passo a passo vão ajudá-lo a instalar o VirtueMart e personalizar o seu template. Com instruções detalhadas você não terá problemas com essas tarefas.

Carrinho Ajax

Este modulo permite os usuarios revisarem os produtos do seu carrinho, sem ter que recarregar a pagina. E ele também oferece uma opção de eliminação rápida de produtos do carro.

Carrossel de Produtos

Permite mostrar imagens de produtos em uma galeria de carrosel. Há também opções para editar e mostrar preços, atribuir imagens de produtos, adicionar títulos, botões e descrições para os seus produtos.
