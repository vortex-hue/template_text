Domande frequenti sui temi WordPress di WPBakery

Come installare il plugin per lavorare con i temi WordPress di WPBakery?

Per installarlo, accedi al tuo sito Web, quindi scegli "Plugin" quindi "Aggiungi nuovo". Carica il file zip del plugin. "Installa ora", attendi, quindi "Attiva plug-in". Ora, inizia a lavorarci scegliendo "Pagine", "Aggiungi nuovo". Controlla la dashboard per vedere le impostazioni disponibili.

Puoi aiutarmi a installare i temi WordPress del generatore di pagine WPBakery?

Sì, richiedi il nostro servizio "Installazione del sito Web" o "Installazione del negozio" .

Come assicurarsi che i temi WordPress del generatore di pagine WPBakery siano multilingue?

Controlla i dettagli del prodotto. Se è pronto per WPML, puoi aggiungere una nuova lingua. Puoi anche richiedere il nostro "Servizio sito web multilingue".
