After payment, you receive an email with a link to download the archive and instructions for editing the website's appearance after installation. Follow the link and download the archive. After that, please install it using the site admin panel. To do this, follow a few steps:

  • In the area that will immediately appear, click "Choose File."
  • Afterward, a new window will open. Select the downloaded zip file with the template.
  • Then click the "Install" button.

You have successfully finished the task if you followed these simple steps and the installation started. Once the process is complete, activate the WordPress membership site template.
